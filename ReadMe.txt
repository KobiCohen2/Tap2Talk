In order to run the project:
1. Download the project using the command "git clone" and the address in the document.
2. Open the project with Android Studio.
3. Connect android device to the computer with API 26 or higher.
4. Allow to debug with the android device in settings->developer options->USB debugging.
5. Run the project in Android Studio by pressing the green triangle.
6. Choose your device to run the project on it, and press OK.
